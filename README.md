# Blog Drafts: AI Blogs and Tutorials

This repository contains drafts, code, and supporting assets for blog posts and tutorials about
AI, LLMs, and related engineering topics.

Each post lives in its own numbered directory under `blogs/` (e.g.
`blogs/004_building_a_geospatial_monitoring_system_for_payment_terminals/`), with the article
draft, any supporting code, and a post-specific `README.md` covering setup and usage.

## Working with this repo

- Posts are written in Markdown; any Markdown previewer works (e.g., VS Code's built-in preview).
- Some posts include runnable code or scripts - see each post's own `README.md` for prerequisites
  and how to run them.

## Contributing / Feedback

Suggestions and corrections are welcome. Open an issue or PR with clear context.

## Notes

Content is a work in progress and may change. Scripts are intended for local
experimentation - review and adapt before running.
